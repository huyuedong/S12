forloop.counter是从1开始计数，forloop.counter0是从0开始计数

    {% for user_obj in user_objs %} {% if forloop.counter > 2 %}
  • username: {{ user_obj.name }}, age:{{ user_obj.age }}
    • {% else %}
  • username: {{ user_obj.name }}, age:{{ user_obj.age }}
    • {% endif %} {% endfor %}

等于有两种写法，1.5之后支持 == ，以前都用ifequal

    {% for user_obj in user_objs %} {# {% ifequal forloop.counter 2 %}#} {% if forloop.counter == 2 %}
  • username: {{ user_obj.name }}, age:{{ user_obj.age }}
    • {% else %}
  • username: {{ user_obj.name }}, age:{{ user_obj.age }}
    • {# {% endifequal %}#} {% endif %} {% endfor %}

偶数行背景为红色，注意|两边不能有空格

    {% for user_obj in user_objs %} {% if forloop.counter|divisibleby:"2" %}
